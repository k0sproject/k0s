## k0s kubeconfig admin

Display Admin's Kubeconfig file

### Synopsis

Print kubeconfig for the Admin user to stdout

```shell
k0s kubeconfig admin [command] [flags]
```

### Examples

```shell
k0s kubeconfig admin > ~/.kube/config
export KUBECONFIG=~/.kube/config
kubectl get nodes
```
